# UserMoon

Лендинг для Telegram-бота UserMoon с интерактивной 3D-луной.

## Структура

- `index.html` — основная страница
- `style.css` — стили
- `moon3d.js` — 3D-сцена (Three.js)
- `main.js` — логика интерфейса
- `protect.js` — защита от копирования

## Особенности

- Реалистичная 3D-луна с процедурными кратерами и фазами
- Интерактивное освещение, реагирующее на скролл
- Частицы с физикой вокруг луны
- Три режима ответов бота (Обычная/Затмение/Кровавая)
- Защита исходного кода
- Адаптивный дизайн

## Запуск

Открыть `index.html` в браузере или развернуть на любом хостинге.

## Зависимости

- Three.js (CDN)
